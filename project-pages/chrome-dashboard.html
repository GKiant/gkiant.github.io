Chrome dashboardextension

HTML5, CSS3, Javascript

GitHub Repo Live version

Built using HTML5, CSS3, and Javascript, this extension provides you with a wealth of information at your fingertips. Featuring stunning photos from Unsplash API, you will get random images for your background and stay up-to-date on the weather with OpenWeather API. Plus, you can track your favorite cryptocurrencies with Coingecko API, all in one convenient location. Whether you're checking the weather, keeping an eye on the markets, or just looking for a new desktop background, this dashboard extension has got you covered!
